## 1. ブランドコンセプト

### ブランドエッセンス

> **「AIと共に、コードで遊べ。」**

### ブランドパーソナリティ

1. **先進的（Cutting-Edge）** — AI時代の最前線で、新しい開発スタイルを体現する
2. **クリエイティブ（Creative）** — コードを「書く」から「奏でる」へ。遊び心と実験精神
3. **オープン（Open）** — 誰でも参加できる。知識を共有し、共に成長する文化

### ブランドストーリー

Live Vibe Coding Clubは、AIとともにコードを書く新しい開発スタイル「Vibe Coding」を実践するエンジニアコミュニティである。ライブコーディングを通じてAIの可能性を探求し、創造性と技術が交差する場を創り出す。私たちは、コードを単なる成果物ではなく、リアルタイムの表現行為として捉える。

### 差別化ポイント

AIを道具としてではなく **共創パートナー** として捉え、そのプロセスをライブで共有する唯一のエンジニアコミュニティ。

## 4. トーン & マナー

### トーンの3原則

1. **テクニカルだが遊び心がある** — 技術的正確さを保ちつつ、堅くなりすぎない。コードは真剣に、雰囲気は軽やかに。
2. **簡潔でエネルギッシュ** — 冗長な説明より、パンチのある短文。ライブ感を文章でも体現する。
3. **オープンでインクルーシブ** — 初心者を排除しない。専門用語を使うときは自然に文脈で伝える。

### DO / DON'T リスト

#### DO（こうする）

1. **能動態で書く** — 「AIと一緒にコードを書こう」
2. **具体的な技術名を出す** — 「Claude Codeで自動生成」「Cursorでペアプロ」
3. **カジュアルな敬語を使う** — 「〜してみましょう」「〜ですね」
4. **行動を促す短いフレーズを使う** — 「今すぐ参加」「コードで遊ぼう」
5. **コミュニティの実績や事例を共有する** — 実際のライブコーディング成果を見せる

#### DON'T（こうしない）

1. **過度にフォーマルな表現を使わない** — ×「ご参加賜りますようお願い申し上げます」
2. **AIの脅威論を煽らない** — ×「AIに仕事を奪われる前に」
3. **排他的な表現を使わない** — ×「上級者限定」「プロだけの空間」
4. **曖昧な修飾語を多用しない** — ×「すごい」「最高の」「革命的な」
5. **長文の段落を書かない** — 1段落3文以内。箇条書きを活用する

### 文体ガイド

| 項目 | ルール |
|------|--------|
| 一人称 | 「私たち」または省略。「弊社」「当クラブ」は使わない |
| 敬語レベル | です・ます調。ただしSNSではタメ口もOK |
| 専門用語 | Vibe Coding、AI、LLM等はそのまま使用。初出時に簡潔な補足を添える |
| 数字 | アラビア数字を使用。「3つのステップ」「100人以上」 |
| 英語混在 | 技術用語は英語のまま。カタカナ変換しない（×コミット → ○commit） |

## 5. ロゴコンセプト

### ロゴタイプの方向性

**コンビネーションマーク** — シンボル + ワードマークの組み合わせ。シンボル単体でもアイコンとして機能する。

### モチーフ提案

#### 案A: 「Wave Pulse」

音波・パルスをモチーフにした抽象的な波形。ライブコーディングの「リアルタイム性」と音楽的な「Vibe」を表現。3色グラデーションを波形に適用。

#### 案B: 「Code Orbit」

`< >` のコードブラケットが軌道を描くように配置されたシンボル。コードとAIが循環・共創する様子を表現。中心にドット（AIの目）を配置。

#### 案C: 「Spark Node」

ニューラルネットワークのノードが発光しているイメージ。AIとの接続点・起点を象徴。既存ロゴの丸いフォルムと親和性が高い。

### 使用ルール

| 項目 | 仕様 |
|------|------|
| 最小使用サイズ（Web） | 24px × 24px（シンボルのみ）/ 120px幅（コンビネーション） |
| 最小使用サイズ（印刷） | 8mm × 8mm（シンボルのみ）/ 40mm幅（コンビネーション） |
| クリアスペース | シンボルの高さの25%を全方向に確保 |
| カラーバリエーション | フルカラー（グラデーション）/ 単色白 / 単色黒 |

### NG使用例

- ロゴの縦横比を変更する
- グラデーションの色順を変更する
- ロゴに影やアウトラインを追加する
- 低コントラストの背景にフルカラーロゴを配置する
- ロゴの一部を切り取って使用する
